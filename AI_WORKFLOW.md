## Prompts & Workflow

### 1. Understanding the requirements
> Summarize the Edstruments assessment email and explain what I need to build.

Used to clarify scope: config-driven filter system in React + TypeScript + Vite.

### 2. Project implementation
> Build it with clean, professional code.

Scaffolded the Vite + React + TypeScript project with MUI, filter engine, reusable components, mock API, and sample datasets.

### 3. Code review
> Check if the code is clean and professional.

Reviewed architecture, separation of concerns, and filtering logic.

### 4. Repository cleanup
> Remove AI-specific files and make the project look professional.

Rewrote README, removed submission-only files, cleaned git history.
